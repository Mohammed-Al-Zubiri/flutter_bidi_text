# flutter_bidi_text Example

A sample application demonstrating the flutter_bidi_text package.

## Getting Started

This example demonstrates how to use the `flutter_bidi_text` package to create text widgets that automatically detect and set text direction based on content.

### Running the Example

1. Clone the repository
2. Navigate to the example directory
3. Run the application:

```bash
cd example
flutter pub get
flutter run
```

## Features Demonstrated

- **BidiText Widget**: Automatically displays text in the correct direction based on content
- **BidiTextField**: Text field that adjusts direction automatically as you type
- **BidiTextFormField**: Form field with auto-direction detection and validation
- **Sample Length**: Control how much text is analyzed to determine direction

## Usage Notes

- Try typing in different languages to see the direction change automatically
- Notice how the direction indicator updates in real-time
- Experiment with different sample lengths to see how they affect direction detection
